# Купика — Списки покупок

Простой статический проект на HTML/CSS/JS. Основной файл `kupika.html` содержит всю разметку и скрипты приложения.

## Подготовка к загрузке на GitHub

1. Инициализуйте репозиторий:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
2. Создайте удалённый репозиторий на GitHub и подключитесь:
   ```bash
   git remote add origin <URL-репозитория>
   git push -u origin main
   ```

## Развёртывание на Vercel

Файл `vercel.json` настроен для статического сайта и использует `index.html` как входную страницу. Если у вас всё ещё есть `kupika.html`, переименуйте его в `index.html` или сохраните оба файла.

1. Установите [Vercel CLI](https://vercel.com/docs/cli) (по желанию):
   ```bash
   npm install -g vercel
   ```
2. В папке проекта выполните:
   ```bash
   vercel  # следуйте инструкциям
   ```

В раскладке репозитория рекомендуется держать `index.html` в корне, поскольку Vercel автоматически обслуживает этот файл из корня без дополнительных маршрутов.

## Структура

```
kupika/
├── index.html         # главный HTML-файл приложения (переименован из kupika.html)
├── kupika.html        # (по желанию) исходник, можно удалить
├── README.md          # это описание
├── .gitignore         # файлы, исключаемые из репозитория
└── vercel.json        # конфигурация для Vercel
```
